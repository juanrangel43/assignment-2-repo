How to Use JavaScript in a Webpage

  1. Linking an External Script

    To add an external script to your HTML file, simply Use the script tag as follows.

    <script scr="path/to/my/script.js"></script>

  2. How to Write JavaScript within an HTML file

    To add JavaScript code directly into your HTML file, you can do so by adding the code between <script> tags without adding anything to the src value.

    <script>
    window.addEventListener("load", () => {
    console.log("This function is executed once the page is fully loaded");
    });
    </script>
